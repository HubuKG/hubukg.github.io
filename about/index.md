# {% include icon.html icon="fa-solid fa-building-columns" %}团队简介

湖北大学教育智能计算与应用团队（Intelligent Computing and Application on Education Lab，ICAE）依托计算机学院建设，旨在促进人工智能、教育智能计算的相关基础研究的发展和人才培养。

团队负责人李志飞，理学博士。湖北大学计算机学院副教授、硕士生导师，入选湖北省级人才计划，兼任湖北大学本科生院副院长（挂职）。毕业于华中师范大学国家数字化学习工程技术研究中心，主要从事知识图谱、推荐系统、智能问答等方向研究。先后主持国家自然科学基金青年项目和湖北省自然科学基金面上项目，近五年在TKDE、TMM、TNNLS、TKDD、INFFUS、AAAI、中国科学:信息科学、电子学报等期刊或会议发表论文40余篇，其中ESI高被引论文3篇。获湖北省科技进步奖二等奖1项、2025 IP&M Reviewer Awards，授权发明专利、软件著作权10余项。CCF高级会员、IEEE/ACM 会员，担任TKDE、TNNLS、TKDD等20余个期刊及会议审稿人和AAAI、EMNLP等会议领域主席。
